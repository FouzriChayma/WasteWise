Voici la liste d'attributs de mes entités : Stock et StoreHouse

Entité Stock :

•	id_st : Identifiant 

•	name_st : Nom ou description du type de déchet stocké (par exemple, "Papier", "Plastique", "Verre", etc.).

•	quantity_st : La quantité de déchets stockée en KG

•	date_d'ajout_st : La date à laquelle les déchets ont été ajoutés au stock.

•	date_dernière_mise_à_jour_st : La date de la dernière mise à jour de la quantité de déchets dans le stock.

•	statut_st : Le statut actuel du stock de déchets (par exemple, " Available ", "in shipping ", "En cours de traitement", etc.).

•	#id_sh : L'emplacement physique où les déchets sont stockés.



Entité StoreHouse :

•	id_sh: Identifiant 

•	name_sh: Nom de l'entrepôt 

•	location_sh: Localisation de l'entrepôt 

•	capacity_sh: Capacité de stockage de l'entrepôt 

•	date_created_sh: Date de création de l'entrepôt.

•	status_sh: Statut de l'entrepôt (e.g., "Active", "Inactive", "Under Maintenance", etc.).
